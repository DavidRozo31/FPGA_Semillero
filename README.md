# FPGA Semillero - Clases

Repositorio base para dictar clases de FPGA con una ruta ordenada de estudio.

## Estructura

- `notebooks/`: notebooks por clase (ordenados por prefijo numérico).
- `codigo/`: ejemplos de Verilog organizados por tema/clase.

## Ruta sugerida de clases

1. **Clase 01 - Introducción a FPGA**  
   Notebook: `notebooks/01_introduccion_fpga.ipynb`  
   Código asociado: `codigo/01_compuertas_basicas/and_gate.v`

2. **Clase 02 - Verilog combinacional**  
   Notebook: `notebooks/02_verilog_combinacional.ipynb`  
   Código asociado: `codigo/02_sumador_4bits/adder4.v`

3. **Clase 03 - Lógica secuencial**  
   Notebook: `notebooks/03_logica_secuencial.ipynb`  
   Código asociado: `codigo/03_maquina_estados/fsm_led.v`

4. **Clase 04 - Mini proyecto UART**  
   Notebook: `notebooks/04_proyecto_uart.ipynb`  
   Código asociado: `codigo/04_uart_basica/uart_tx_stub.v`

## Cómo usar este repositorio

1. Abre cada notebook siguiendo el orden numérico.
2. Explica la teoría de la clase.
3. Revisa y ejecuta el ejemplo de código asociado en `codigo/`.
4. Extiende cada módulo como ejercicio práctico para la clase.
